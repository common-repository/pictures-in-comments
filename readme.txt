== Description ==

Pictures in Comments (PIC) provided a upload form for images in the post respond's in your WP's blog.

== Installation ==

1. Upload `pictures-in-comments/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set config data in `config.php`
